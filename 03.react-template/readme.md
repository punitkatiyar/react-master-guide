#  Apply CSS In Recat Component .

## CSS Inline Method 
```
function Hello() {
    let head = {
        color: "white",
        fontSize: "2rem",
        padding: "2% 5%",
        backgroundColor: "teal",
    }
    return (
        <>
            {/* <div style={{ color: "white", fontSize: "2rem", padding: "2% 5%" }}>Hello</div> */}
            <div style={head}>Hello</div>
        </>
    )
}

export default Hello
```
